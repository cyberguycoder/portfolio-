Employee Management System

My original project for academic and practical use. Built by me to manage employees, tasks, performance, and admin workflows in one web app.

Live Demo / GitHub: GitHub Repository

Overview

Developed Dec 2024 – Feb 2025 as an academic project. Built with core PHP, MySQL and CRUD operations to manage employees, tasks and evaluations.

Features

  • Admin users: create, update, delete employee records
  • Assign tasks to employees and monitor status
  • Performance evaluations and task history
  • Responsive UI with modern layout and clean navigation

Technologies

PHP, MySQL, HTML, CSS, Bootstrap
