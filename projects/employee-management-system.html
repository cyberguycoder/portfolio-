Employee Management System

This project is a live Employee Management System built with PHP, MySQL, Bootstrap, and JavaScript. It includes employee record management, salary tracking, and easy CRUD operations.

Project Overview

  • Fully functional employee dashboard with list, add, edit, and delete operations
  • Modern dark/professional design with Bootstrap 5
  • Easy setup using PHP built-in server and MySQL database
  • Perfect for portfolio showcase and live demo
Open Live System

NOTE: This app requires a local PHP server. If the file downloads instead of opening, start the server first (for example, run php -S localhost:8000 from the app folder), then open the URL.

About The Project

Employee Management System is developed to simplify employee handling and organizational workflow management.

The system allows administrators to manage employee information, assign tasks, track attendance, and monitor performance through a secure dashboard.

It helps organizations improve productivity, reduce manual paperwork, and maintain employee records digitally.
